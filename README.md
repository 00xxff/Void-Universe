前言

每当我手里握着着两块强磁把玩，然后不停的快速靠近分开靠近，陶醉的感受其中巨大的斥力和吸力的同时，双手却没有其他的感觉，我就清楚的知道，这个世界至少是4维的。然而每每当我思考这些现象的原理并尝试去探寻其力量的本质和根源时，各种资料最终都会指向狭义相对论，这一直领我十分苦恼，因为在我看来，时间只是人类创造出来描述事物变化快慢的概念，不是客观存在的，用时间来做物理学的基础是十分不负责任的，是倒反天罡的行为。

我们所认知的时间的标定是需要依赖某些物理量来实现的，而这些物理量在高速运动时受某些因素影响发生了变化，从而导致其时钟频率变慢，这才产生了时间膨胀的错觉(这也是本模型的解释)，物理时钟在变慢，不是指某类用于计时的物理量在变慢，而是指整个物理系统在变慢，就是纯粹字面意义上的所有的粒子相对运动都变慢了，原因就是朝着某个单一方向的惯性动能导致某个关键物理机制变慢了，然后才导致了所有的物理现象都变慢，整个系统就像电脑CPU被降频了一样，这是硬件行为，而在这个硬件上运行的软件系统中是感觉不到时间变慢的，但是通过与外部参考系的第三方相互对比就可以观察到。

所以你是愿意相信时间变长变短玩弄人类这么扯淡的事呢，还是更愿意相信我们的宇宙是在一个大的相对静止的惯性系统里呢，就是虚空是相对静止的，所有物质的移动都是相对虚空的移动。这样才不至于出现两个惯性系统上无法分辨谁是静止谁是移动的悖论，我们只需要对虚空做相对比较就可以了，虚空就是静止的，其他的相对论性表述都是物质与虚空相对移动的结果。

所以个人认为相对论对时空的解释其实是倒果为因，只是使用了结果总结出来的规律来描述同样规律的现象，而现代物理学则完全陷入了哲学陷阱，为了追求结果的正确放弃了对根源的追寻，这种情况已经把基础物理学逼上了绝路，所以我们需要重新审视相对论，需要重新分析造成相对论这种规律性结果的物理根源，而不是一味的使用从结果中得到的变换参数来解释人类尚未能观测到的量子世界。

因此狭义相对论和广义相对论应该只作为一种验证工具，而非作为物理学基础来阻止人类对微观世界的探索。

我们不是要去否定或推翻某个理论或学说，而是要站在这些理论和学说的肩膀上，去看到更远处的世界真相。

我们的目的是找到一个大一统理论，将经典模型、量子力学、相对论都统一起来，同时不能像弦理论那样无限增加纬度，所以我们必须做减法，将所有的物理现象都统一到一个底层框架上，这样即使我们暂时还无法验证这个底层框架的正确性，但只要这个底层框架比经典模型简单，又能统一实现所有的上层架构，那么这个底层框架就是先进的有价值的。

所以我们需要对宇宙做逆向工程，对相对论做逆向工程，对量子力学做逆向工程。
